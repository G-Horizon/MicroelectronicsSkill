# NumDivisions Function

Returns the number of divisions in a vector. Vectors created by multi-step runs such as Monte Carlo are sub-divided into divisions with one division per step. For a full explanation of this concept, see  [Multi-division Vectors](scriptlanguage_accessingsimulationdata.htm#scriptlanguage_accessingsimulationdata__multidivisionvectors) .
