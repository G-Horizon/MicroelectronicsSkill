# GetSelectedGrid Function

Returns id of selected Grid. Grids are sections of a graph that contains one or more horizontally stacked Y-axes and one or more vertically stacked X-axes.

Each axis is linked to a single grid which may be read from the Grid property.

### Arguments

No arguments

### Returns

Return type: string
