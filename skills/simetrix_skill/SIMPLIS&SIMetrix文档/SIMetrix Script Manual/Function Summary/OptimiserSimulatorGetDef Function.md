# OptimiserSimulatorGetDef Function

Returns an optimiser ID for the most recent simulator's optimiser run. The returned ID can be used for any optimiser function. Note that the ID captures the current state of the simulator's optimiser definition

### Arguments

No arguments

### Returns

Return type: Real

Optimiser ID
