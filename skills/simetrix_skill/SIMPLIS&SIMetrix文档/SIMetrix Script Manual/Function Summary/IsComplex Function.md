# IsComplex Function

Returns 1 (=TRUE) if the supplied argument is complex and 0 (=FALSE) if the argument is any other type.

### Arguments

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **Number** | **Type** | **Compulsory** | **Default** | **Description** |
| 1 | any | Yes |  | Vector |
