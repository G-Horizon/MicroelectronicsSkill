# SupportedWriteFormats Function

SIMetrix schematics and graphs Save Picture features can write the displayed image to a graphical file. This function returns the formats supported. See  [CopyClipSchem](com_copyclipschem.htm)  and  [CopyClipGraph](com_copyclipgraph.htm)  for commands that can generate image files.

### Arguments

No arguments

### Returns

Return type: string array
