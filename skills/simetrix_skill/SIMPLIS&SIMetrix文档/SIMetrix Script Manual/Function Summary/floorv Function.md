# floorv Function

Returns the argument truncated to the next lowest integer. Same as  [Floor](func_floor.htm)  , except that it also accepts vector inputs, for example:

```
Floorv([3.45, 7.89, -3.45]) = [3, 7, -4]
```

### Arguments

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **Number** | **Type** | **Compulsory** | **Default** | **Description** |
| 1 | real | Yes |  | vector |

### Returns

Return type: real vector

Returns a vector of the arguments truncated to next lowest integers
