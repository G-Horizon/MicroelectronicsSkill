# OptimiserSimulatorGetInfo Function

Get simulator optimiser file information

### Arguments

No arguments

### Returns

Return type: String array

|  |  |
| --- | --- |
| 0 | Path to attached schenmatic |
| 1 | Path to results file |
| 2 | Path to definition file |

Returns an empty vector if no simulator optimiser is current
