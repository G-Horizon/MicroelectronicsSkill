# CleanPath Function

Cleans a file path by reducing ".." where possible and eliminating ".". Also returns the path using a consistent directory separator.

### Arguments

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **Number** | **Type** | **Compulsory** | **Default** | **Description** |
| 1 | string | Yes |  | File path |

### Returns

Return type: string

Cleaned path
