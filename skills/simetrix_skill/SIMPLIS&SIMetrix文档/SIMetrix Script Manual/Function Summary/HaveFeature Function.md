# HaveFeature Function

Detrmines whether a specified license feature is available.

### Arguments

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **Number** | **Type** | **Compulsory** | **Default** | **Description** |
| 1 | string | Yes |  | Feature name |

#### Argument 1

Name of license feature. Currently may be one of, 'basic', 'advanced', 'micron', 'rtn', 'simplis\_if', 'AD', 'schematic' or 'scripts'.

### Returns

Return type: real

Returns 1.0 if the license feature is available otherwise it returns 0.0.
