# ExistFile Function

Tests whether the given file exists. Does not test whether the file can be opened. Use  [CanOpenFile](func_canopenfile.htm)  to test if a file exists and can also be opened.

### Arguments

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **Number** | **Type** | **Compulsory** | **Default** | **Description** |
| 1 | string | Yes |  | File path |

### Returns

Return type: real

1.0 if file exists otherwise 0.0
