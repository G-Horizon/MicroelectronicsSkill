# GetGraphObjPropValues Function

Returns property values for the specified object. The function will return an array containing all property values. The order of the values corresponds to the return value of  [GetGraphObjPropNames](func_getgraphobjpropnames.htm) .

See  [Graph Objects](applications_graphobjects.htm)  for more information.

See  [GetGraphObjPropValue](func_getgraphobjpropvalue.htm)  to obtain one property at a time.

### Arguments

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **Number** | **Type** | **Compulsory** | **Default** | **Description** |
| 1 | string | Yes |  | Graph object ID |

### Returns

Return type: string array

|  |  |
| --- | --- |
| **Index** | **Description** |
| 0 |  |
