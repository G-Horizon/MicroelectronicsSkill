# ReadSIMPLISF11Data Function

Returns information about the current SIMPLIS simulation configuration stored in the schematic's F11 window.

### Arguments

No arguments

### Returns

Return type: string array

Array of semi-colon delimited strings. Each string defines a SIMPLIS simulation option setting. The first field is the name of the setting. The second is the type (boolean,integer,real,enum) and the third is the current value.
