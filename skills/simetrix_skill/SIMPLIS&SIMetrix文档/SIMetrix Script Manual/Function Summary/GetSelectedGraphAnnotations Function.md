# GetSelectedGraphAnnotations Function

Returns the ID for all selected graph annotation objects. If no objects are selected, the function returns '-1'. If no graphs are open, the function returns an empty vector.

See  [Graph Objects](applications_graphobjects.htm)  for information on graph annotation objects.

### Arguments

No arguments

### Returns

Return type: string
