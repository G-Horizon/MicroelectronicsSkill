# ACSourceDialogStr Function

Displays dialog box intended for the user definition of an AC source. Argument is a string array with two elements which specify the initial values for the two controls as follows:

|  |  |
| --- | --- |
| 0 | Magnitude |
| 1 | Phase |

The function returns a string array of length 2 with the same format as the argument described above.

### Arguments

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **Number** | **Type** | **Compulsory** | **Default** | **Description** |
| 1 | string array | Yes |  | Initial value |

### Returns

Return type: String array
