# phase Function

Returns the phase of the argument in degrees. Identical to  [ph](func_ph.htm) .

Each of the functions  [ph](func_ph.htm)  , phase and  [phase\_rad](func_phase_rad.htm)  produce a continuous output i.e. it does not wrap from 180 degrees to -180 degrees.

This function always returns a result in degrees. This has changed from versions 3.1 and earlier which returned in degrees or radians depending on the setting of the 'Degrees' option. For phase in radians, use  [phase\_rad](func_phase_rad.htm) .
