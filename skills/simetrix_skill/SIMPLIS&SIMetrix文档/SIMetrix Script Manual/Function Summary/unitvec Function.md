# unitvec Function

Returns a vector consisting of all 1's. Argument specifies length of vector.

### Arguments

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **Number** | **Type** | **Compulsory** | **Default** | **Description** |
| 1 | real | Yes |  |  |

#### Argument 1

Number of elements in result

### Returns

Return type: real array
