# OptimiserLoadWidgetFromXML Function

Loads an optimiser widget (GUI) from an XML file

### Arguments

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **Number** | **Type** | **Compulsory** | **Default** | **Description** |
| 1 | String | Yes |  | Filename |

#### Argument 1

Path to file to load into widget

### Returns

Returns 1 if successful, 0 if file failed to load, -1 if there is no current valid optimiser widget
