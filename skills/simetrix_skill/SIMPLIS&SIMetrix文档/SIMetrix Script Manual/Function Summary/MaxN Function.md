# MaxN Function

Returns an array equal to the length of each argument. Each element in the array holds the larger of the corresponding elements of each argument. A maximum of 32 arguments may be supplied. All arguments must be the same length; an empty vector will be returned if this is not the case.

All arguments must be real.

### Arguments

No arguments

### Returns

Return type: real array
