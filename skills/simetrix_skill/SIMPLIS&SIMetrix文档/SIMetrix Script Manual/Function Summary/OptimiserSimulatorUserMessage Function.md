# OptimiserSimulatorUserMessage Function

Returns a string providing the current state of the simulator's optimiser for display to the user. The message includes the current iteration number, parameter values and resulting measurements

### Arguments

No arguments

### Returns

String giving current iteration number, parameter values and resulting measurements. Empty if there is no current simulator optimiser
