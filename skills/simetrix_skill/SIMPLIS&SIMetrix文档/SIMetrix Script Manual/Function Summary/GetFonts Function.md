# GetFonts Function

Returns the names of all objects in the program whose font may be edited. The function is usually used in conjunction the function  [GetFontSpec](func_getfontspec.htm)  , the function  [SelectFontDialog](func_selectfontdialog.htm)  and the command  [EditFont](com_editfont.htm) .

### Arguments

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **Number** | **Type** | **Compulsory** | **Default** | **Description** |
| 1 | string | Yes |  | options |

#### Argument 1

If set to 'supportcolour' will return only fonts that have an editable colour.

### Returns

Return type: string array
