# Val Function

Returns argument converted to a value. The conversion assumes that the string supplied is an expression.

### Arguments

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **Number** | **Type** | **Compulsory** | **Default** | **Description** |
| 1 | any | Yes |  | Input value |

### Returns

Return type: real/complex
