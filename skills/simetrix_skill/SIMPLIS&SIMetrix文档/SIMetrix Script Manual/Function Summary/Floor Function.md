# Floor Function

Returns the argument truncated to the next lowest integer. Examples:

```
Floor(3.45) = 3
Floor(7.89) = 7
Floor(-3.45) = -4
```

This function accepts only scalar input values. See  [floorv](func_floorv.htm)  for a version that accepts vector input.

### Arguments

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **Number** | **Type** | **Compulsory** | **Default** | **Description** |
| 1 | real | Yes |  | scalar |

### Returns

Return type: real
