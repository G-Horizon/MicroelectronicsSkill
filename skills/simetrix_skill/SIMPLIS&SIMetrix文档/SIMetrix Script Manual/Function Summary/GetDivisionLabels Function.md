# GetDivisionLabels Function

Get the division labels for a specified graph curve. Division labels are assigned to multi-step data and described the name and value of the item that was swept.

### Arguments

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **Number** | **Type** | **Compulsory** | **Default** | **Description** |
| 1 | String | Yes |  | Curve id |

#### Argument 1

Curve id

### Returns

Return type: String array

List of values usually in the form name=vale
