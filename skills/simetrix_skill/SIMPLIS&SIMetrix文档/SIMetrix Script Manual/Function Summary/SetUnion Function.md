# SetUnion Function

Finds the union of two sets of strings, that is the set of strings that are in either argument. No string will be duplicated in the result even if duplicated in one of the inputs. This function accepts empty vectors for each argument

### Arguments

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **Number** | **Type** | **Compulsory** | **Default** | **Description** |
| 1 | string array | Yes |  | First set of strings |
| 2 | string array | Yes |  | Second set of strings |

#### Argument 1

First set of strings

#### Argument 2

Second set of strings

### Returns

Return type: string array

Union of the two arguments
