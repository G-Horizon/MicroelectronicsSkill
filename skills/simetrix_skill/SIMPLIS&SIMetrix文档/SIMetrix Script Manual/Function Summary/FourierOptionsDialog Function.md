# FourierOptionsDialog Function

Same as  [DefineFourierDialog](func_definefourierdialog.htm)  except that only the Fourier sheet is displayed. The remaining tabbed sheets are hidden.

### Arguments

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **Number** | **Type** | **Compulsory** | **Default** | **Description** |
| 1 | string array | Yes |  | initial values |
| 2 | real array | No |  | sample vector |

### Returns

Return type: string array
