# SymbolLibraryManagerDialog Function

Opens the Symbol Library Manager dialog box. See  [User's Manual/Symbol Editor and Library/Symbol Library Manager](../../user_manual/topics/symboleditorlibrary_symbollibrarymanager.htm)  for details of this feature.

### Arguments

No arguments

### Returns

Return type: string array

|  |  |
| --- | --- |
| **Index** | **Description** |
| 0 | User operation: |  |  | | --- | --- | | 0 | Close button pressed | | 1 | Place button pressed | | 2 | Edit button pressed | |
| 1 | Internal name of selected symbol |
| 2 | Full path of selected library file |
| 3 | Empty - reserved for future use. |
