# CloseFile Function

Closes a file opened using  [OpenFile](func_openfile.htm) .

### Arguments

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **Number** | **Type** | **Compulsory** | **Default** | **Description** |
| 1 | real | Yes |  | File handle |

#### Argument 1

File handle to close. This is the value returned by the  [OpenFile](func_openfile.htm)  function.

### Returns

Return type: real
