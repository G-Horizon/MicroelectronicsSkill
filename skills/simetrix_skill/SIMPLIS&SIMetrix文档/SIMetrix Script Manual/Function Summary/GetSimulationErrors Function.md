# GetSimulationErrors Function

Returns all errors raised by the most recent simulation.

### Arguments

No arguments

### Returns

Return type: string array

Returns a string array with all errors raised by the most recent simulation. If the simulation ran correctly with no errors, an  [empty value](scriptlanguage_variablesconstantstypes.htm#scriptlanguage_variablesconstantstypes__emptyvalues)  will be returned. Note that this function only returns error messages; it does not return warnings.
