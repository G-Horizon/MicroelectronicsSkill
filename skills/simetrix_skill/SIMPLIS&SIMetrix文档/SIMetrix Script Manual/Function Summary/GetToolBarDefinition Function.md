# GetToolBarDefinition Function

Returns names of all buttons on a specified toolbar.

### Arguments

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **Number** | **Type** | **Compulsory** | **Default** | **Description** |
| 1 | string | Yes |  | Toolbar name |

#### Argument 1

Name of toolbar as defined in command  [CreateToolBar](com_createtoolbar.htm)  and populated using  [DefineToolBar](com_definetoolbar.htm) .

### Returns

String array of names of buttons that have been added to the specified toolbar using the  [DefineToolBar](com_definetoolbar.htm)  command.
