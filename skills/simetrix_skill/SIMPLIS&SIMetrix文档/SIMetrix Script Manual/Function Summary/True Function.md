# True Function

Returns TRUE (1) if the vector specified by name in argument 1 exists AND is nonzero. If argument 2 is set to 'SearchCurrent', the current group as well as the local and global groups will be searched for the vector, otherwise only the local and global groups will be searched. See  [Groups](scriptlanguage_accessingsimulationdata.htm#scriptlanguage_accessingsimulationdata__groups)  for an explanation of groups.

### Arguments

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **Number** | **Type** | **Compulsory** | **Default** | **Description** |
| 1 | string | Yes |  | Vector name |
| 2 | string | No | <<empty>> | Option |

### Returns

Return type: real
