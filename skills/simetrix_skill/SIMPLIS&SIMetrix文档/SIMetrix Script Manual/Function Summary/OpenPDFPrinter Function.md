# OpenPDFPrinter Function

Sets up printing for PDF output.

### Arguments

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **Number** | **Type** | **Compulsory** | **Default** | **Description** |
| 1 | String | Yes |  | Filename |
| 2 | String | No | A4 (use option DefaultPDFpageSize to override) | Page size |

#### Argument 2

Page size. May be one of A3, A4, A5, Letter, Legal or Tabloid

### Returns

Return type: String

Success or Failed message.
