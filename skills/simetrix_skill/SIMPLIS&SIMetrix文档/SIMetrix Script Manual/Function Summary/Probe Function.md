# Probe Function

Changes schematic cursor to a shape depicting an oscilloscope probe. Returns when the user presses a mouse key. If the left key is pressed return value is 1 otherwise it is 0. Probe returns on both up and down strokes of mouse key. See  [Cross probing](scriptlanguage_atutorial.htm#scriptlanguage_atutorial__example3)  for an example of using the Probe function.

### Arguments

No arguments

### Returns

Return type: real

1 if left button clicked, 0 if cancelled (right button or escape)
