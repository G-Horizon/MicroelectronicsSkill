# integ Function

Integrates the argument with respect to its reference. See  [Vector References](scriptlanguage_accessingsimulationdata.htm#scriptlanguage_accessingsimulationdata__vectorreferences)  for details.

The function uses simple trapezoidal integration.

An error will occur if the argument supplied has no reference.

### Arguments

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **Number** | **Type** | **Compulsory** | **Default** | **Description** |
| 1 | real array | Yes |  | vector |

### Returns

Return type: real array
