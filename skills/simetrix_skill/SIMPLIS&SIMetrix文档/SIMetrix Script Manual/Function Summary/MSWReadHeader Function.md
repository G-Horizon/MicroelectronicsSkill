# MSWReadHeader Function

Read header information of an MSW file. The header contains the analysis line used to create it along with the simulation group

### Arguments

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **Number** | **Type** | **Compulsory** | **Default** | **Description** |
| 1 | String | Yes |  | File to read |

#### Argument 1

File to read

### Returns

Return type: String array

Header information in order:

analysis-line, groupname, date, title.
