# abs Function

Returns absolute value or magnitude of argument. This function is identical to the function  [mag](func_mag.htm) .

### Arguments

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **Number** | **Type** | **Compulsory** | **Default** | **Description** |
| 1 | real/complex array | Yes |  | vector |

### Returns

Return type: real array

Absolute value of input
