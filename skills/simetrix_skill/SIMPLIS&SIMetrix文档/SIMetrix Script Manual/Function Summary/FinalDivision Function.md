# FinalDivision Function

Returns final division of a multi-division vector. If the vector is single division the function will simply return the vector

### Arguments

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **Number** | **Type** | **Compulsory** | **Default** | **Description** |
| 1 | Real/complex | Yes |  | Input vector |

#### Argument 1

Input vector

### Returns

Return type: Same as arg 1

Final division of input vector
