# RefName Function

Returns the name of the reference of the supplied vector. See  [Vector References](scriptlanguage_accessingsimulationdata.htm#scriptlanguage_accessingsimulationdata__vectorreferences) . Note that the function  [Ref](func_ref.htm)  returns the actual data for the reference.

| --- | --- | --- | --- | --- |
| **Number** | **Type** | **Compulsory** | **Default** | **Description** |
| 1 | real/complex array | Yes |  | Vector |
