# Pan Command

Pan (scroll) schematic specified number of grid squares.

```
Pan <x> <y>
```

### Parameters

|  |  |
| --- | --- |
| x | Movement in x direction. A positive value moves the schematic to the left. |
| y | Movement in y direction. A positive value moves the schematic up. |
