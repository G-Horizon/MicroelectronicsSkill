# MakeAlias Command

Converts a string variable to an alias.

An alias is a string representing a numeric expression. For more information see  [Aliases](scriptlanguage_expressions.htm#scriptlanguage_expressions__aliases) .

```
MakeAlias <vector-name>
```

### Parameters

|  |  |
| --- | --- |
| vector-name | Variable to be converted |
