# ConvertBinaryGraph Command

Converts a graph binary file created with version 8.1 or earlier, into the XML format used by version 8.2 and later.

```
ConvertBinaryGraph <file name> <out-path>
```

### Parameters

|  |  |
| --- | --- |
| File to convert | File to convert |
| File to receive XML converted result | File to receive XML converted result |
