# SelectSchematic Command

Focuses on the specified schematic. Use either id or path.

```
SelectSchematic /id <schematic-id> | <schematic path>
```

### Parameters

|  |  |
| --- | --- |
| /id | Specifies that the input argument is a schematic ID. This can be obtained from  [OpenSchematic](func_openschematic.htm)  or  [GetSchematicTabs](func_getschematictabs.htm) . |
