# Focus Command

Focus on a window.

Only one of the options can be used at a time.

```
Focus [/named <window-name>] [/userid <window-id>] [schem|graph]
```

### Parameters

|  |  |
| --- | --- |
| schem|graph | Currently or most recently selected schematic or graph window receives input focus. |
