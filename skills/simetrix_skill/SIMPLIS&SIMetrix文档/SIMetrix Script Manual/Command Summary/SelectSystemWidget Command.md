# SelectSystemWidget Command

Selects the specified system widget. If two or more system widgets are arranged in a tabbed window, this command will select the tab for the specified systsm widget. System widgets are: "Command Shell" "File View" "Part Selector"

```
SelectSystemWidget <view-context-name>
```
