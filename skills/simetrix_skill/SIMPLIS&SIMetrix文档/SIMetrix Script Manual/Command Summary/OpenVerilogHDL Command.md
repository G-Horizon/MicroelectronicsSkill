# OpenVerilogHDL Command

Opens a Verilog-HDL source file in the text editor. This will apply syntax highlighting for the Verilog-HDL language.

```
OpenVerilogHDL <filename>
```

### Parameters

|  |  |
| --- | --- |
| /encoding | encoding. For details see documentation of second argument to  [LoadFile](func_loadfile.htm) |
| /fws | File watcher status, enable|disable|auto |
| filename | Path of Verilog-HDL source file to open |
