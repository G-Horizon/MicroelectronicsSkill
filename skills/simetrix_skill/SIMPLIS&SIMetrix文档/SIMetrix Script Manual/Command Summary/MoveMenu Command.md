# MoveMenu Command

Moves the position of a menu item by a specified count.

```
MoveMenu [/bypos position] <menu-path> <shift-by>
```

### Parameters

|  |  |
| --- | --- |
| /bypos | Value  *position*  is an optional number that identifies a menu item by its position within a sub-menu. If this is specified the menupath must identify a sub-menu rather than a menu item. |
| menu-path | Path to the menu to move, see  [DefMenu](com_defmenu.htm)  for full details of path names. |
| shift-by | Number of positions by which menu is moved. A positive number moves the menu down, a negative number moves it up. |
