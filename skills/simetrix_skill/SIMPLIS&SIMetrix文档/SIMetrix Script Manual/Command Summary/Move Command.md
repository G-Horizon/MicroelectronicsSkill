# Move Command

Initiates the schematic move operation. User interactive command.

### Parameters

|  |  |  |  |  |  |  |  |
| --- | --- | --- | --- | --- | --- | --- | --- |
| /mode | Specifies editing mode to use for move operation. Options are:  |  |  | | --- | --- | | "default" | Use option setting "SchematicMoveMode" | | "ClassicMove" | Basic rubberbanding mode | | "Orthogonal" | Wires edited so that they remain at right angles as much as possible | |
