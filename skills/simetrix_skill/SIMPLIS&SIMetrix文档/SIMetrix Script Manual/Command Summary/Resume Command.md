# Resume Command

Resumes a previously paused simulation.

### Parameters

|  |  |
| --- | --- |
| /maxruntime | Specifies the maximum run time (that is real time not analysis time). If exceeded, the simulation will automatically enter a paused state |
| /stopTime | If specified, will change the transient stop time to the new value. This can be longer or shorter than the original. If a the run has already reached the time specified, the setting will be ignored and the stop time will remain at its current value |
