# HideCurve Command

Hides specified curve.

```
HideCurve <curve-id>
```

### Parameters

|  |  |
| --- | --- |
| curve-id | Id of curve to hide. Curve id is returned by the functions  [GetSelectedCurves](func_getselectedcurves.htm)  ,  [GetAxisCurves](func_getaxiscurves.htm)  and  [GetAllCurves](func_getallcurves.htm) . |
