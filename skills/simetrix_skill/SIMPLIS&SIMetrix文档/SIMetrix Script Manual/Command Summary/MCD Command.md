# MCD Command

Makes a directory and sets it as current (same as  [MD](com_md.htm)  followed by  [Cd](com_cd.htm)  ).

```
MCD <directory-name>
```

### Parameters

|  |  |
| --- | --- |
| directory-name | Name of directory to be created. |
