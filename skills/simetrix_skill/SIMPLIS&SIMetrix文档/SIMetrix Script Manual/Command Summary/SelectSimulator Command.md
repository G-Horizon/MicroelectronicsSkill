# SelectSimulator Command

Selects current simulator for selected schematic.

```
SelectSimulator simulator name
```

### Parameters

|  |  |
| --- | --- |
| simulator name | Name of simulator to be selected. Current valid values are "SIMetrix" and "SIMPLIS". |
