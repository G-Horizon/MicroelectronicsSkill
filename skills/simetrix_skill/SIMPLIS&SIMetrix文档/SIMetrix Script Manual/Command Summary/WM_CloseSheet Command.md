# WM\_CloseSheet Command

This will close the most recently accessed or explicitly-specified content window. A content window may be a schematic, symbol editor, graph, text editor or welcome page.

```
WM_CloseSheet [<content-widget-name>]
```

### Parameters

|  |  |
| --- | --- |
| content-widget-name | Name of content widget to close as returned by  [WM\_GetContentWidgetNames](func_wm_getcontentwidgetnames.htm) |
