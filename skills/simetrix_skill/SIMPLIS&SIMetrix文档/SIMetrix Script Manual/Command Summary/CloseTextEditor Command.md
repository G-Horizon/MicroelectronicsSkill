# CloseTextEditor Command

Closes the currently selected text editor based widget.

### Parameters

|  |  |
| --- | --- |
| /type | Optional. Specifies the type of editor to close. Options are:  *LogicDefinitionEditor*  ,  *NetlistEditor*  ,  *ScriptEditor*  ,  *TextEditor*  ,  *VerilogAEditor*  ,  *VerilogHDLEditor* . |
