# SetRef Command

Attaches  *reference-expression*  to  *vector-name* . Previous reference is detached and deleted if no longer used. See  [Expressions](scriptlanguage_expressions.htm)  for details on references.

```
SetRef <vector-name> <reference-expression>
```
