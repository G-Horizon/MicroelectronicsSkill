# CurveEditCopy Command

Copy specified curves to the internal clipboard. Curves so copied may be subsequently plotted using the command  [Curve](com_curve.htm)  with the /icb switch.

```
CurveEditCopy <curve-id> [<curve-id> ...]
```

### Parameters

|  |  |
| --- | --- |
| curve-id | Id of curve. A number of functions return this value including  [GetSelectedCurves](func_getselectedcurves.htm) . |
