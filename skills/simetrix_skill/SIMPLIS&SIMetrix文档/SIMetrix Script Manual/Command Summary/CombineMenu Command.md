# CombineMenu Command

Combines several menus into a separate menu.

```
CombineMenu <menu1|menu2|...> <new_menu_name>
```

### Parameters

|  |  |
| --- | --- |
| menus | Set of menu names to merge, separted by '|'. |
| new\_menu\_name | The name of the new menu to merge to. |
