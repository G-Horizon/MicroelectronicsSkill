# EditFont Command

Changes the spec for the named font object.

```
EditFont <font-name> <font-spec>
```

### Parameters

|  |  |
| --- | --- |
| font-name | Name of font object. This can be any of the names returned by the function  [GetFonts](func_getfonts.htm) . (These are listed when the menu **File** > **Options** > **Font...** is selected.) |
| font-spec | Text string that defines the font. The functions  [GetFontSpec](func_getfontspec.htm)  and  [SelectFontDialog](func_selectfontdialog.htm)  return font spec values. |
