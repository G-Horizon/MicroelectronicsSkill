# CopyFile Command

Copies a file.

```
CopyFile [/force] <from-file> <to-file>
```

### Parameters

|  |  |
| --- | --- |
| /force | If specified,  *to-file*  will be overwritten if it exists. Otherwise if  *to-file*  exists, the command will fail. |
| from-file | Source file |
| to-file | Destination file |
