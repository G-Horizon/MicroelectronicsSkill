# UpdateSymbol Command

Updates symbols on currently selected schematic from global symbol library.

Schematics store local copies of any symbols that it uses. If the copy of that symbol in the global library is modified, the schematics own copy is unaffected. This command causes the specified symbol to be updated from global library. See  [How Symbols are Stored](applications_schematicsymbolscriptdefinition.htm#applications_schematicsymbolscriptdefinition__howsymbolsstored) .

```
UpdateSymbol <symbol-name>
```

### Parameters

|  |  |
| --- | --- |
| symbol-name | Name of the symbol to be updated. |
