# AddImage Command

Adds an image to the current schematic. This is an interactive action and will attach the image to the cursor until it is placed on the schematic.

```
AddImage [/dimension <target_size>] <filename>
```

### Parameters

|  |  |
| --- | --- |
| /dimension | Target size in grid units squared. Optional. |
| /reference | If set, states that filename is actually a reference. |
| filename | The name of the file to import as a full path or reference if the reference flag is set. |
