# MoveFile Command

Moves a file from  *path-1*  to  *path-2* .

```
MoveFile [/force] <path-1> <path-2>
```

### Parameters

|  |  |
| --- | --- |
| /force | If specified,  *path-2*  will be overwritten if it already exists. If not specified, the command will fail if  *path-2*  exists. |
