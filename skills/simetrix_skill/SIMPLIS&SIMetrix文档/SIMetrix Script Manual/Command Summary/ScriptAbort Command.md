# ScriptAbort Command

Aborts execution of script. Note that this command can only be usefully executed from a key or menu item which has been defined with the direct execution option specified (option flag 5 or /immediate switch for  [DefMenu](com_defmenu.htm)  ). See  [User Defined Key and Menu Definitions](applications_userinterface.htm#applications_userinterface__userdefinedkeymenu) .
