# SwitchModelSection Command

Switches model library to a new selection. This is intended to be temporary and will be restored automatically to the default in a new session. Use SwitchModelSection /default to restore to default setting.

```
SwitchModelSection /default | <section-name>
```

### Parameters

|  |  |
| --- | --- |
| /default | Restore to default section |
