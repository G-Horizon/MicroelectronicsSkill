# NewBasicTextEditor Command

Creates a new plain text document in the SIMetrix environment. Use this for files with no recognised format. Use one of the following commands to create documents with specific formats:

* [NewNetlist](com_newnetlist.htm)  to create a model file or netlist file
* [NewLogicDefinitionEditor](com_newlogicdefinitioneditor.htm)  to create a logic definition file for the abritrary logic block
* [NewScript](com_newscript.htm)  to create a script
* [NewVerilogA](com_newveriloga.htm)  to create a Verilog-A source file
* [NewVerilogHDL](com_newveriloghdl.htm)  to create a Verilog-HDL source file
