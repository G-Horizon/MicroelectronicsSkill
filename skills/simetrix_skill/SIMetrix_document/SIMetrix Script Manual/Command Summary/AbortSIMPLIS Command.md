# AbortSIMPLIS Command

Aborts the current simulation. Abort performs the same action as  [Pause](com_pause.htm)  followed by  [Reset](com_reset.htm) . It stops the current run and then deletes all data associated with it except for any simulation vectors.

Note that this command can only be executed by an assigned key or menu with the direct execution option specified.
