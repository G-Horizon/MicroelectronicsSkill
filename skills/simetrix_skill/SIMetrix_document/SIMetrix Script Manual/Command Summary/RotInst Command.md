# RotInst Command

Changes orientation of selected items.

```
RotInst [<orientation>]
```

### Parameters

|  |  |  |  |  |  |  |  |  |  |  |  |  |  |  |  |  |  |
| --- | --- | --- | --- | --- | --- | --- | --- | --- | --- | --- | --- | --- | --- | --- | --- | --- | --- |
| orientation | Integer from 0 to 7 to specify how symbol should be oriented:  |  |  | | --- | --- | | 0 | No change | | 1 | Rotate clockwise 90° | | 2 | Rotate clockwise 180° | | 3 | Rotate clockwise 270° | | 4 | Mirror through vertical axis | | 5 | Mirrored + 90°Rotation | | 6 | Mirrored + 180°Rotation | | 7 | Mirrored + 270°Rotation | |
