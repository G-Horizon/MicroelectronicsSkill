# Zip Command

Zips a file or a directory of files. If the path to a directory is given, all sub-directories are recursively zipped to the archive too.

```
Zip <archive-file-path> <source-path>
```

### Parameters

|  |  |
| --- | --- |
| archive-file-path | The path to the archive file to create. This will overwrite anything that already exists. |
| source-path | The path to the source file or directory to recursively archive. |
