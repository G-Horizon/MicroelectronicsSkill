# RepeatLastMenu Command

Executes the menu most recently selected by the user. SIMetrix remembers the last command executed for each top level menu and this menu must be specified with this command.

```
RepeatLastMenu <window-name> <top-menu-name>
```

### Parameters

|  |  |
| --- | --- |
| window-name | Identifies the window type that owns the menu. See  [DefMenu](com_defmenu.htm)  for list of possible values. |
| top-menu-name | The top level menu name. This is the name that appears in the menu bar. |
