# UnLet Command

Destroy vector.

```
UnLet <vector-name> ...
```

### Parameters

|  |  |
| --- | --- |
| vector-name | Name of vector to be destroyed. Unless the vector is in the  *user*  group, the vector's full qualified name must be used. |
