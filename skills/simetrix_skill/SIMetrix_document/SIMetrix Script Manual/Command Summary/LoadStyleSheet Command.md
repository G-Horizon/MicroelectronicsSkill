# LoadStyleSheet Command

Applies a style sheet to the whole application.

### Parameters

|  |  |
| --- | --- |
| /clear | Clears the current style sheet first. |
| /direct | Loads style sheet directly from input string. |
| /file | Loads style sheet from a file. |
