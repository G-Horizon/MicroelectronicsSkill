# OptimiserRequestFinish Command

Requests that current optimisation analysis is terminated. Optimiser will complete after end of current analysis unless /abort is specified in which case it will abort immediately

### Parameters

|  |  |
| --- | --- |
| /abort | Aborts immediately instead of completing current analysis |
