# DrawArrow Command

Draws an arrow in the schematic editor, as specified by the  *loc*  parameter..

```
DrawArrow [/loc <x1> <y1> <x2> <y2>]
```

### Parameters

|  |  |
| --- | --- |
| /loc | Defines the positioning of the arrow as 4 integer values relating to position within the schematic: x1, y1, x2, y2. |
