# RenameMenu Command

Renames a menu item.

```
RenameMenu <menu-path> <new-item-name>
```

### Parameters

|  |  |
| --- | --- |
| menu-path | The full path of the menu to change the name for. |
| new-item-name | The new name to use for the menu item. |
