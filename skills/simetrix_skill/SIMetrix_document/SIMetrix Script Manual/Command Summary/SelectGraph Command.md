# SelectGraph Command

Switches the graph tabbed sheet to the graph specified by  *graph-id* .

```
SelectGraph <graph-id>
```

### Parameters

|  |  |
| --- | --- |
| /focuswin | If specified, the window containing the specified graph will be brought into focus. |
