# Plot Command

Plot can be used to add a new curve to an existing graph created with Plot or to change the way it is displayed.

The Plot command is identical to the  [Curve](com_curve.htm)  command except that it always creates a new graph sheet unless the current sheet is empty. The Curve command never creates a new graph sheet.

For full details refer to  [Curve](com_curve.htm)  command.
