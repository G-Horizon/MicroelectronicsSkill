# NewGrid Command

Creates a new grid. See  [User's Manual/Graphs,Probes and Data Analysis/Graph Layout](../../user_manual/topics/graphs_probesanddataanalysis_graphlayout.htm)  for more information on axes and grids.

### Parameters

|  |  |
| --- | --- |
| /detach | Make a detached x-axis. A detached x-axis is not connected to other axes and will zoom independently |
| /digital | Make a digital grid |
