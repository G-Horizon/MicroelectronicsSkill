# ListModels Command

Generates a dictionary of all models and subcircuits currently available to the simulator (e.g. installed with menu **File** > **Model Libraries** > **Add/Remove Libraries** see  [User's Manual/Device Library and Parts Management](../../user_manual/topics/devicelibraryandpartsmanagement_partsmanagement_installingmodels.htm)  ). Result is written to  *filename* . A single line will be produced for each model or subcircuit found containing the device name, its type (NPN, JFET, subcircuit etc.) and the filename in which it was found along with the line number.

```
ListModels <filename>
```
