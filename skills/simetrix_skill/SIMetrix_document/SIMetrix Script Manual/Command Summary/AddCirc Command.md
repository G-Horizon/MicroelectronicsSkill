# AddCirc Command

AddCirc is a Symbol Definition Command. Creates a circle.

```
AddCirc <x-org> <y-org> <radius>
```

### Parameters

|  |  |
| --- | --- |
| x-org | x co-ordinate of circle centre |
| y-org | y co-ordinate of circle centre |
| radius | radius of circle |
