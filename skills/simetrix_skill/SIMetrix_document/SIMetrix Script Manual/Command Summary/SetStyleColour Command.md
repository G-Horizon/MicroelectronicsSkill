# SetStyleColour Command

Sets the style with the specified colour. The colour is specified as a hex colour code, in blue-green-red format using `0x00bbggrr`.

### Parameters

|  |  |
| --- | --- |
| /colour | The new colour to apply using a blue-gree-red hex code. |
| /stylename | The name of the style to change the colour for. |
