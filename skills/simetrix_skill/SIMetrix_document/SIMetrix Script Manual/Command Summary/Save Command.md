# Save Command

Saves the currently selected schematic.

```
Save [/all]
```

### Parameters

|  |  |
| --- | --- |
| /all | If specified, all open schematics will be saved. Note that schematics that have not previously been saved will not be saved by this command. |
| /ascii | If specified the schematic will be saved in ASCII format unconditionally |
| /nostyle | If specified, the style library will not be written to the schematic. |
| /xml | If specified the schematic will be saved in XML format unconditionally |
