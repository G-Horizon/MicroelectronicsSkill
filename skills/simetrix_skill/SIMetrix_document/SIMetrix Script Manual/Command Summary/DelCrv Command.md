# DelCrv Command

Deletes the specified curve or curves on the selected graph.  *curve\_id*  is returned by the functions  [GetSelectedCurves](func_getselectedcurves.htm)  ,  [GetAxisCurves](func_getaxiscurves.htm)  and  [GetAllCurves](func_getallcurves.htm) .

Optionally a curve name may be specified. This must be the whole text of the curve legend. It is the value returned by the function  [GetCurves](func_getcurves.htm) .

```
DelCrv <curve-id>|<curve-name> [...]
```
