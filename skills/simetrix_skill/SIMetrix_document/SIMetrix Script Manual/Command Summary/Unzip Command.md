# Unzip Command

Unzips a zip archive.

```
Unzip <archive-file-path> <target-directory>
```

### Parameters

|  |  |
| --- | --- |
| archive-file-path | The path to the archive file to unzip. |
| target-directory | The path to target directory to unzip the content to. This will overwrite anything that already exists with the same names. |
