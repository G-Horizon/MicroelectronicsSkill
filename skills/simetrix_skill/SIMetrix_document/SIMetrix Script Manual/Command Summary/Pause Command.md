# Pause Command

Pauses current simulation (if any). Note that this command can only be executed by assigning it to a key or menu item with the direct execution option specified (option flag 5). For more information see  [User Defined Key and Menu Definitions](applications_userinterface.htm#applications_userinterface__userdefinedkeymenu) .

A paused simulation can be restarted with the command  [Resume](com_resume.htm) .
