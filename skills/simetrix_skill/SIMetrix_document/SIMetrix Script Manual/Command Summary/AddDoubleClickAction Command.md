# AddDoubleClickAction Command

Adds a double click action to the selected elements. Must provide both a type and a command for the action.

### Parameters

|  |  |
| --- | --- |
| /command | The argument used when performing the type of action selected. |
| /type | The type of double click action. Options are:  |  |  | | --- | --- | | OpenSchematic | Opens the schematic defined by the command. | | OpenWebPage | Opens the webpage defined by the command. | | RunScript | Runs the script defined by the command. | |
