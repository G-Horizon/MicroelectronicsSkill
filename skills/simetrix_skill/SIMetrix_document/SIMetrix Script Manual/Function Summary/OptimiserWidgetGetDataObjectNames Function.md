# OptimiserWidgetGetDataObjectNames Function

Returns names of all data objects in the current optimiser widget (GUI). Name can be used as an argument to  [OptimiserWidgetGetDataObject](func_optimiserwidgetgetdataobject.htm)  to get actual data.

### Arguments

No arguments

### Returns

Return type: String array

List of names
