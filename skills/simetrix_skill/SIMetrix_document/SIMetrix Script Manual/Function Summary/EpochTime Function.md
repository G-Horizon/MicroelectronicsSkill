# EpochTime Function

Returns the number of seconds elapsed since midnight, January 1, 1970.

### Arguments

No arguments

### Returns

Return type: real

### Notes

The return value has a numerical resolution of 1 ms but the useable resolution is system dependent and usually much coarser.
