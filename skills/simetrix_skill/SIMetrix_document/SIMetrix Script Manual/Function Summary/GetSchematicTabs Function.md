# GetSchematicTabs Function

Returns IDs for all open schematics. The ID is an integer value that uniquely identifies a schematic and may be used by a number of commands and functions to perform operations on a schematic. For more information, refer to the  [OpenSchematic](func_openschematic.htm)  function.

### Arguments

No arguments

### Returns

Return type: real array

### See Also

* [GetOpenSchematics](func_getopenschematics.htm)  returns the corresponding paths of open schematics
* [OpenSchematic](func_openschematic.htm)  for more information on schematic IDs
