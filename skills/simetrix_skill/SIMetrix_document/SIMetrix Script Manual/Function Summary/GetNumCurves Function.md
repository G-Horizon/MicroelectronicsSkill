# GetNumCurves Function

Returns the number of curves in a curve group. This is applicable to curves plotted for a multi-step analyses such as Monte Carlo. Note that all non-digital curves plotted from multi-step analyses are grouped

### Arguments

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **Number** | **Type** | **Compulsory** | **Default** | **Description** |
| 1 | string | Yes |  | Curve ID |

### Returns

Return type: real
