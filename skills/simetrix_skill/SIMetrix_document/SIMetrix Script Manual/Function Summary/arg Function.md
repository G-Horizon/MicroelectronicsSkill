# arg Function

Returns the phase of the argument in degrees. Unlike the functions  [phase](func_phase.htm)  and  [phase\_rad](func_phase_rad.htm)  , this function wraps from 180 to -180 degrees. See  [arg\_rad](func_arg_rad.htm)  function below for a version that returns phase in radians.

### Arguments

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **Number** | **Type** | **Compulsory** | **Default** | **Description** |
| 1 | real/complex array | Yes |  | vector |

### Returns

Return type: real/complex array

Returns the arc tangent of the argument. Result is in degrees.
