# WM\_GetCentralWidgetGeometry Function

Returns window geometry information for the editor region of the window.

### Arguments

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **Number** | **Type** | **Compulsory** | **Default** | **Description** |
| 1 | string | Yes |  | Window name |

#### Argument 1

Window name as returned from the function  [WM\_GetWindowNames](func_wm_getwindownames.htm) .

### Returns

Return type: string

Geometry information for the editor region.
