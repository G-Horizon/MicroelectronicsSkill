# LoadSensitivityReport Function

Each sensitivity analysis generates an XML file containing the results of that analysis. This includes the evaluation of the goal functions and and the component values used for each case. This function displays that data in a table view

### Arguments

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **Number** | **Type** | **Compulsory** | **Default** | **Description** |
| 1 | String | Yes |  | Input file |

#### Argument 1

Sensitivity XML file path

### Returns

Return type: real

Status: success: 1.0, fail: 0.0
