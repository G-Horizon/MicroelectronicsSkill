# CloseSchematicTab Function

Closes a schematic using its ID. A schematic's ID may be obtained from  [OpenSchematic](func_openschematic.htm)  or  [GetSchematicTabs](func_getschematictabs.htm) .

Function returns 1 if successful otherwise returns 0.

### Arguments

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **Number** | **Type** | **Compulsory** | **Default** | **Description** |
| 1 | real | Yes |  | Schematic ID |

### Returns

Return type: real
