# phase\_rad Function

Returns the phase of the argument in radians. Identical to  [ph](func_ph.htm)  , except the result is in radians.

Produces a continuous output i.e. it does not wrap from 180 degrees to -180 degrees.

### Arguments

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **Number** | **Type** | **Compulsory** | **Default** | **Description** |
| 1 | real/complex array | Yes |  | vector |

### Returns
