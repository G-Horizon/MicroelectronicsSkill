# CopyDivisionData Function

>Copies division data from a vector to another vector. Division data are text items attached to vectors that typically define information on multi-step simulations.

### Arguments

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **Number** | **Type** | **Compulsory** | **Default** | **Description** |
| 1 | Real or complex array | Yes |  | Vector to copy division data from |
| 2 | Real or complex array | Yes |  | Vector to receive division data |

#### Argument 1

Source vector for division data. This is expected to be a multi-division vector

#### Argument 2

Target vector for division data. This is expected to be a multi-division vector

### Returns

Return type: Same as argument 2

Vector from argument 2 with division data copied form argument 1
