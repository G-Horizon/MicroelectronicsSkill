# OptimiserCreateFromXML Function

Create an optimiser definition from an XML file. The optimiser definition including its parameters, analyses and measurements can be described using an XML based definition. This function loads such a definition from a file and returns a reference ID.

### Arguments

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **Number** | **Type** | **Compulsory** | **Default** | **Description** |
| 1 | String | Yes |  | Filename of XML definition file |

#### Argument 1

Filename of XML definition file

### Returns

Return type: Real

ID of optimiser definition, -1 on failure
