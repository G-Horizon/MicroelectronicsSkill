# GetLastGraphObjectAdded Function

Returns ID of last graph object added. The type of object can also be specified, using one of the object type names, where the last object of that type will be returned.

### Arguments

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **Number** | **Type** | **Compulsory** | **Default** | **Description** |
| 1 | string | No |  | Object type |

#### Argument 1

The type of object to search for the last added element of.
