# OptimiserWidgetCreateOptDef Function

Returns an ID referencing the current optimiser widget (GUI)

### Arguments

No arguments

### Returns

Return type: Real

ID to reference contents of currently highlighted optimiser widget (GUI). Returns -1 if no widget is currently open.
