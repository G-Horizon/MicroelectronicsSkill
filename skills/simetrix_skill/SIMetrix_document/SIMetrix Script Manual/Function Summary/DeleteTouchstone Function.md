# DeleteTouchstone Function

Deletes data loaded by  [LoadTouchstone](func_loadtouchstone.htm)

### Arguments

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **Number** | **Type** | **Compulsory** | **Default** | **Description** |
| 1 | Real | Yes |  | ID of Touchstone data |

#### Argument 1

ID of Touchstone data as returned by  [LoadTouchstone](func_loadtouchstone.htm) . If 0, will delete all available Touchstone data

### Returns

Return type: Real

Number of data items deleted.
