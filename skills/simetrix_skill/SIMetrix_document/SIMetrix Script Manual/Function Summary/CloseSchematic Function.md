# CloseSchematic Function

Closes a schematic handle opened using  [OpenSchematic](func_openschematic.htm) . Schematic handles are used to obtain information about schematics that are not currently being displayed. For more information see  [OpenSchematic](func_openschematic.htm)

Function returns 1 if successful otherwise returns 0.

### Arguments

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **Number** | **Type** | **Compulsory** | **Default** | **Description** |
| 1 | real | Yes |  | Schematic handle |

### Returns

Return type: real
