# OptimiserWidgetWriteXML Function

Writes current optimiser widget to the file specified in the argument

### Arguments

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **Number** | **Type** | **Compulsory** | **Default** | **Description** |
| 1 | String | Yes |  | Filename |

#### Argument 1

Path to file to receive XML definition

### Returns

Return type: Real

Return status: Success:1, write failed:0, No widget open:-1
