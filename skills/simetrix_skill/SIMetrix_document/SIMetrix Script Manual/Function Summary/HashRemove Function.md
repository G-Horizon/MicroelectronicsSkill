# HashRemove Function

### Arguments

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **Number** | **Type** | **Compulsory** | **Default** | **Description** |
| 1 | real | Yes |  | Hash table id as return by |
| 2 | string | Yes |  | List of keys |

### Returns

Return type: real

Number of items deleted
