# WAV\_CloseFile Function

Close a WAV file opened with  [WAV\_OpenFile](func_wav_openfile.htm)

### Arguments

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **Number** | **Type** | **Compulsory** | **Default** | **Description** |
| 1 | Real | Yes |  | WAV file handle |

#### Argument 1

WAV file handle obtained from  [WAV\_OpenFile](func_wav_openfile.htm)

### Returns

Return type: Real

1, success, 0 fail - means that the hanlde is not valid
