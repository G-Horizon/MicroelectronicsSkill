# GetXAxis Function

Returns id of selected x-axis. This function has identical behaviour to  [GetSelectedXAxis](func_getselectedxaxis.htm) . In version 8.1 and earlier, a graph could only have a single x-axis and this function returned its ID. For clarity we recommend that new code use the  [GetSelectedXAxis](func_getselectedxaxis.htm)  function.

### Arguments

No arguments

### Returns

Return type: string
